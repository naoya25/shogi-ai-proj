# 学習データについて

学習データはAI同士の対局を行いそれを使うことにする
使うAIは、alphabeta法(depth3)+駒得評価 の単純なAIで1000局(15時間くらいかかった)

評価値は、勝ち+1、負け-1、引き分け±0

常に手番側視点で
| 局面 | 結果 | 教師 |
| --- | ---- | -- |
| 先手番 | 先手勝ち | +1 |
| 後手番 | 先手勝ち | -1 |
